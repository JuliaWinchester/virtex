# virtex

[![virtex3d on npm](https://img.shields.io/npm/v/virtex3d.svg?style=flat)](https://www.npmjs.com/package/virtex3d)

three.js object viewer.

##Setup

Clone the repo then run:

    npm install
    gulp
    gulp test
